intent-bench

Intent Fulfillment Benchmark for Coding Agents

Does providing structured intent to coding agents improve implementation effectiveness? intent-bench measures this with a controlled A/B design: agents receive identical task prompts, but the treatment condition additionally receives structured requirements via a pluggable intent layer.

On the baseline task (url-shortener), the intent layer adds overhead without improving completion -- this is expected and shows the cost of structured planning on trivially-sized tasks. On the standard task (task-manager), completion rate increases from 20% to 80% -- the intent layer prevents the thrashing loops that cause agents to fail on complex dependency chains.
